Small pieces of Scala that replace whole frameworks.

Interesting classes
===================
 - com.n6.ews.bb.Blackboard: Actor based blackboard implementation. Concurrent and non-persistent.
 - com.n6.ews.ib.Topic: Pub-sub with Actors.

Requires
========
 - Scala 2.7.5
 - Specs 1.2.5
 - Ant
 - Ivy
